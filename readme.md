## How to run
1. Install Maven
1. Run in the console:
    ```sh
    mvn package
    java -cp target/menu-ordering-system-1.0-SNAPSHOT.jar com.evive.mealordering.Application
    ```
1. Enter meal information:
    ```sh
    Enter MENU: 
    Breakfast 1,2,3
    Eggs, Toast, Coffee
    ```
